# Environment

## neovim
- Install [neovim](https://github.com/neovim/neovim/wiki/Installing-Neovim)

## vim-plug
- Install [vim-plug](https://github.com/junegunn/vim-plug#neovim)

## init.vim
```shell
mkdir -p ~/.config/nvim
touch ~/.config/init.vim
```

```vim
" ~/.config/init.vim

call plug#begin()

call plug#end()
```
